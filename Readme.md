## ğŸš€ Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn
- Git

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Purab2001/DocFlow.git
cd DocFlow
```

2. Install dependencies:
```bash
npm install
# or

```

3. Start the development server:
```bash

```
